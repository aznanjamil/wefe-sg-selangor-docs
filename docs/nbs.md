# NBS — Nature-Based Solutions

The **NBS** page catalogues specific nature-based interventions evaluated for the Sungai Selangor basin — things like riparian buffers, recharge wetlands, peatland rehabilitation, and reforestation that can deliver water, climate, and biodiversity benefits at the same time.
